### Introduction:
=========
1. Evision的轻量化版本.<br>
2. 界面基于IMGUI.<br>
3. 更完美的跨平台和编译体验.<br>
4. 效率提升.<br>
5. 可扩展的框架,便于理解和添加功能.<br>
6. 前后端分离,便于只需要算法的朋友浏览代码,UI逻辑不入侵算法.<br>
