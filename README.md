*本项目致力于构建一个轻量化,图形化,高效率的机器视觉Demo,目的是为机器视觉初学者提供一份良好的入门参考代码.*

### 项目规约
   1. ui采用PySide2-Widget模式编写,界面定义使用ui文件.
   2. 界面逻辑使用python编写.
   3. 有性能要求的算法使用C++实现,输出动态链接库.
